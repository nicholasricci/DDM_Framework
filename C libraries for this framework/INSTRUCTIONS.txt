INPUT

to get extents in your program use this:
DDM_Get_Extents(int argc, char* argv);

to get dimension in your program use this:
DDM_Get_Dimension(int argc, char* argv);

to get alfa in your program use this:
DDM_Get_Alfa(int argc, char* argv);

OUTPUT

to write an output coherent to framework use this:
DDM_Write_Result(int argc, char* argv, double total_time);
